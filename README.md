# SmartHire – Resume & JD Aware AI Interview Simulator

SmartHire simulates real technical interviews by analyzing a candidate’s resume and the job description.
It asks adaptive questions and generates a readiness report.

## Features
- Resume & Job Description analysis
- Adaptive interview questions
- Answer validation & feedback
- Final readiness score and report
- Local data processing (privacy-friendly)

## Tech Stack
- Python
- Streamlit

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
